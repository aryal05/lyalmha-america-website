# Blog Banner Images

Store hero banner images for blog posts in this folder.

**Naming Convention:**
- Use descriptive names with hyphens
- Example: `banner-festival-celebration.jpg`
- Example: `banner-cultural-heritage.jpg`

**Recommended Specifications:**
- Format: JPG or PNG
- Size: 1200x600px (2:1 ratio) or 1920x1080px (16:9)
- Max file size: 500KB
- Optimize before uploading

**Usage:**
Import in your components:
```javascript
import bannerImage from '@/assets/images/banners/your-banner.jpg'
```

These banners appear at the top of blog posts and in blog cards.
