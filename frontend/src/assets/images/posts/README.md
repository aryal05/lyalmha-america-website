# Blog Post Images

Store individual blog post images in this folder.

**Naming Convention:**
- Use descriptive names with hyphens
- Example: `newari-festival-indra-jatra.jpg`
- Example: `traditional-food-yomari.jpg`

**Recommended Specifications:**
- Format: JPG or PNG
- Size: 800x600px (4:3 ratio)
- Max file size: 500KB
- Optimize before uploading

**Usage:**
Import in your components:
```javascript
import postImage from '@/assets/images/posts/your-image.jpg'
```
